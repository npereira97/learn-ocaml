Demo Exercise: Working with Lists

This exercise serves as a demo of the style checker functionality we have implemented as an extension to Learn-OCaml. The template on the left contains examples of common style mistakes we have seen students make in their code due to misunderstandings about the code constructs in question.

The style checker provides two categories of feedback: warnings and suggestions. Warnings are what we feel should always be taken into account, while a student may or may not want to apply a suggestion depending on concerns such as code readability.

Since this exercise is only meant as a demo for the style checker, the questions in this exercise are not graded for correctness.

  1. Write a function member: 'a -> 'a list -> bool which checks if an element exists in a list.

  2. Write a function length: 'a list -> int which returns the length of a list.

  3. Write a function filter: ('a -> bool) -> 'a list -> 'a list which returns all the elements of a list satisfying a given predicate. The order of the elements in the list should be preserved.

  4. Write a function num_multiples: int -> int list -> int such that num_multiples x l returns the number of elements in l that are evenly divisible by x. Use length, filter, and divides (as given to you in the prelude) in your answer.
